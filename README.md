# RIFT / ROW

RIFT / ROW is a compact clothing brand concept built around modular city layers, garment-dyed cotton, washed denim, and repairable trims.

## Brand Position

- Name: RIFT / ROW
- Drop: Weathered Utility
- Offer: outerwear, knit layers, and denim that can be worn as a small repeatable wardrobe
- Tone: quiet, tactile, practical, design-led
- Palette: ink, paper, sage, clay, harbor blue, plum, and muted gold

## Files

- `index.html` contains the storefront content and layout.
- `styles.css` contains the full responsive visual system.
- `script.js` handles product filtering and the bag drawer.

Open `index.html` in a browser to view the site.
